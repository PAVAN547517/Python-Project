# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/PAVAN547517/Python-Project.git
```

Go into project

```bash
cd Python-Project
```

---

# ⚙️ Install Requirements

```bash
pip install -r requirements.txt
```

---

# ▶️ Run

```bash
python main.py
```

or

```bash
python filename.py
```
